Form properties and methods

Method

form.submit() method will submit the form automatically.

Form can be submitted manually the user employing a button or input elemnts with a type attibute of submit, or even an iput elemnet with a type attribute of image:

<button type='submit'>Submit</button>
<input type='submit' value='Submit'>
<input type='image' src='button.png'>

form.reset() method will reset allt eh form controls back to their initial values specific in the HTML.

<button type='reset'>Reset<button>

Properties

The form.action property can be used to set the action attribute of a form, so it’s sent to a different URL to be processed on the server:

form.action = '/an/other.url'
